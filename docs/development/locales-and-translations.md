#  Locales and translations

Currently, the app is translated to:

* 🇬🇧 English
* 🇫🇷 French
* 🇩🇪 German
* 🇧🇪 Dutch
* 🇭🇺 Hungarian
* 🇮🇹 Italian
* 🇵🇹 Portuguese
* 🇸🇪 Swedish
* 🇨🇳 Simplified Chinese

If you want to see a new locale added, please  [open a new issue](https://github.com/dreeveapp/dreeve/issues/new?template=translations-and-localisation.md).
Only do this if you are willing to help on the actual translation 🙃.
